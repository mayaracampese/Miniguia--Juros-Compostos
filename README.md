# Miniguia de Estudo – Juros Compostos

## Descrição do Projeto
Este projeto é um miniguia de estudo sobre **juros compostos**, elaborado utilizando o **NotebookLM** como ferramenta de aprendizagem ativa. O objetivo é organizar informações de diversas fontes abertas, gerar resumos estruturados, criar um glossário de conceitos e desenvolver perguntas estratégicas para revisão do conteúdo.

O trabalho demonstra como a **IA pode ser utilizada para aprendizado**, aliando curadoria de fontes, pensamento crítico e organização do conhecimento.

---

## Tema
**Juros Compostos – Introdução à Educação Financeira**

---

## Objetivos de Estudo
- Compreender o conceito de juros compostos.  
- Identificar a diferença entre juros simples e compostos.  
- Entender como os juros compostos impactam investimentos de longo prazo.  
- Aprender exemplos práticos de aplicação do conceito.
